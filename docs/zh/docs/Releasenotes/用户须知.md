# 用户须知<a name="ZH-CN_TOPIC_0232332818"></a>

-   openEuler版本号计数规则由openEuler x.x变更为以年月为版本号，以便用户了解版本发布时间，例如openEuler 20.03表示发布时间为2020年3月。
-   [Python核心团队](https://www.python.org/dev/peps/pep-0373/#update)已经于2020年1月停止对Python 2的维护。openEuler社区于2020年12月31日全面停止维护，openEuler 20.03 LTS SP3仅修复Python 2的致命CVE，请您切换到Python 3。
