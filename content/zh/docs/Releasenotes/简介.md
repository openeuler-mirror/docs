# 简介<a name="ZH-CN_TOPIC_0182825778"></a>

## 版本介绍<a name="zh-cn_topic_0182825778_section2199453141213"></a>

openEuler 1.0 的 Base 版本是基础版本，包含了运行最小系统的核心组件。该版本基于 Linux 主线内核 4.19 版本，进行了 bug 修改、漏洞修复等工作，并加入了一些具备创新性的功能特性。
