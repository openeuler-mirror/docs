# 安装指导<a name="ZH-CN_TOPIC_0184808015"></a>

1.  首先需要安装iSulad容器引擎。

    ```
    # yum install iSulad
    ```

2.  安装系统容器依赖包。

    ```
    # yum install isulad-tools authz isulad-lxcfs-toolkit lxcfs
    ```

3.  查看iSulad是否已经启动。

    ```
    # systemctl status isulad
    ```

4.  开启lxcfs和authz服务。

    ```
    # systemctl start lxcfs
    # systemctl start authz
    ```
