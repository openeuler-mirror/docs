# 使用限制<a name="ZH-CN_TOPIC_0184808088"></a>

特权容器为容器提供了所有功能，还解除了设备cgroup控制器强制执行的所有限制，具备以下特性：

-   Secomp不block任何系统调用
-   /sys、/proc路径可写
-   容器内能访问主机上所有设备

-   系统的权能将全部打开
