# 更改属性<a name="ZH-CN_TOPIC_0183005787"></a>

请使用以下命令检查具体属性，比如 mtu：

```
# nmcli connection show id 'Wifi ' | grep mtu
802-11-wireless.mtu: auto
```

使用如下命令更改设置的属性：

```
# nmcli connection modify id 'Wifi ' 802-11-wireless.mtu 1350
```

使用如下命令确认更改：

```
# nmcli connection show id 'Wifi ' | grep mtu
802-11-wireless.mtu: 1350
```
